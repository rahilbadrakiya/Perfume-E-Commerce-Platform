## ğŸš€ Installation & Setup

### 1. Prerequisites
- **XAMPP** (or any environment with PHP and MySQL).

### 2. Database Configuration
1. Start **Apache** and **MySQL** via XAMPP Control Panel.
2. Create a new database named `perfume_shop` in [phpMyAdmin](http://localhost/phpmyadmin).
3. Import the database schema:
   - Go to the **Import** tab.
   - Choose `sql/schema.sql` from the project directory.

### 3. Local Deployment
1. Clone this repository into your XAMPP `htdocs` directory:
   ```bash
   cd c:\xampp\htdocs
   git clone https://github.com/YOUR_USERNAME/perfume-website-demo.git
   ```
2. Open your browser and navigate to:
   `http://localhost/perfume-website-demo`
